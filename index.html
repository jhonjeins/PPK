¿QUE ES?

¿Que es un PPK Kit?

Consiste en un sistema adaptado al dron que permite conectar un receptor GNSS, en algunos casos este sistema puede ser intrusivo o no. El kit varia según el tipo de modulo y el tipo de instalación

  • ¿Que es PPK?

    Post-processing kinematics (PPK) son mediciones del satélite GNSS sin procesar de las estaciones móviles y de referencia, las cuales son almacenadas para su posprocesamiento

  • Modulo M+

    Linea base hasta 10km - Bandas: L1 - Almacenamiento: 8gb

  • Modulo M2

    Linea base hasta 60km - Bandas: L1+ L2 / E5 - Almacenamiento: 16gb

  • Instralación Intrusiva

    Es aquella instalación que requiere realizar modificaciones al dron tanto externas, como internas

  • Instalación NO Intrusiva

    Es aquella instalación que se realiza de manera externa, solo conectando los componentes a los elementos del dron (El kit puede desmontarse)

  • Modulo AGNSS L1/L2

    Linea base hasta 80km - Bandas: L1 + L2 / E5 - Almacenamiento: 16gb
